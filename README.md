# SoloEMDR

Un outil web autonome de stimulation bilatérale (EMDR) pour un usage individuel : suivi d'un souvenir à travailler, échelles de ressenti, séries de stimulation visuelle/audio/haptique, et clôture de séance.

## Utiliser l'outil

Ouvrez simplement [`index.html`](index.html) dans un navigateur, ou visitez la page hébergée si GitHub Pages est activé pour ce dépôt.

Aucune donnée n'est envoyée à un serveur : les souvenirs et réglages sont stockés localement dans le navigateur (`localStorage`).

## Avertissement

Cet outil n'est pas un substitut à un accompagnement par un·e thérapeute EMDR certifié·e. Il est proposé à titre d'aide personnelle.
